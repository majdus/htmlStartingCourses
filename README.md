# Cours Web Interactif

Mini-cours interactif HTML/CSS/JS en une seule page, avec éditeurs intégrés et aperçu en direct via `iframe`.

## Contenu

- Un cours progressif (étapes 0 à 6).
- Des zones de code modifiables pour HTML, CSS et JavaScript.
- Un bouton "Exécuter" pour voir le rendu immédiatement.

## Démarrage rapide

1. Ouvre `CoursWebInteractif.html` dans un navigateur.
2. Modifie le code dans les zones noires.
3. Clique sur "Exécuter" pour voir le résultat.

## Fichiers

- `CoursWebInteractif.html` : la page du cours interactive.
- `Cours_Web_Interactif_Complet.md` : version texte complète du cours.

## Notes

- Tout fonctionne côté client, sans serveur ni dépendances.
- Si tu veux distribuer le cours, il suffit de partager le fichier HTML.
